# Examples

## Register Check

Input:

> Tolong ubah ke bahasa Madura.

Safer response:

> Tanyakan dulu: untuk teman sebaya, orang tua, pengumuman umum, acara resmi, atau konteks Tapal Kuda? Bahasa Madura memiliki variasi dan tingkat kesopanan.

## Review Mode

Input:

> Cek apakah ini terdengar Madura atau cuma Jawa/Indonesia yang diganti kata.

Response shape:

1. Flag Javanese or Indonesian structure that is only decorated with Madurese words.
2. Check important terms against dictionary/grammar sources.
3. Recommend native-speaker review for public use.

## Dialect Caution

For Bawean, Sumenep, Bangkalan, Pamekasan, Sampang, or Tapal Kuda contexts, ask for the local target before rewriting.
