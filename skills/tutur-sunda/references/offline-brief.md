# Offline Brief

## Local Facts

- Purpose: support cautious drafting and review for Bahasa Sunda.
- Region/context: Jawa Barat, Banten, and Sundanese communities elsewhere.
- ISO 639-3: `sun`.
- Glottocode: `sund1252`.
- Source confidence: Level 2 because grammar, dictionary, and Peta Bahasa references exist.
- Bahasa Sunda has speech-level sensitivity. Register mistakes can sound rude or inappropriate.

## Guardrails

- Do not flatten all Sundanese into Bandung-style everyday speech.
- Do not use informal forms for elders, schools, formal events, religious events, or public announcements unless requested.
- Ask whether the target is loma, lemes, public announcement style, or a specific local variety.
- For Banten, Baduy/Kanekes, or border-area contexts, ask for a local sample.
- For public, ritual, legal, educational, or official use, recommend native-speaker review.

## Sample Prompts

- "Use $tutur-sunda to review this Bahasa Sunda draft for politeness."
- "Rewrite this as polite public Sundanese."
- "Check if this is too casual for a school announcement."

## Safe Output Patterns

- First identify audience and speech level.
- For review, flag informal choices and possible level mixing.
- For rewrite, provide a safer public/polite version with a caveat if uncertain.

## Local Example

Request:

> Tolong ubah ke bahasa Sunda.

Safe response shape:

> Tanyakan dulu: untuk teman sebaya, orang tua, pengumuman umum, atau acara resmi? Setelah itu pilih tingkat tutur yang sesuai.
