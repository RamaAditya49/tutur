# Examples

## Target-Language Examples

Neutral Jabodetabek:

> Aku udah cek, tinggal nunggu file final dari tim desain.

Jaksel:

> Basically ini udah oke, tinggal adjust copy dikit biar tone-nya more friendly.

Jaktim/Jakarta casual:

> Gue udah cek, tinggal beresin dikit. Nanti gue kabarin kalau udah kelar.

Bekasi:

> Gue udah otw, agak ketahan macet. Nanti gue update kalau udah deket.

## Jaksel Code-Mix

Input:

> Tolong bilang desainnya sudah bagus, tapi CTA masih perlu diperjelas.

Output:

> Design-nya udah cakep, vibe-nya juga dapet. Tinggal CTA-nya dibuat lebih clear biar orang langsung ngerti next step-nya.

## Jaktim Casual

Input:

> Tolong ingatkan teman untuk kirim file hari ini.

Output:

> Bro, file-nya kirim hari ini ya. Biar nggak mundur lagi, gue tunggu sebelum jam 5.

## Bekasi Commuter Tone

Input:

> Saya akan terlambat karena jalan macet.

Output:

> Gue agak telat ya, ketahan macet. Nanti gue update lagi kalau udah deket.

## Review Mode

Input:

> Literally gue tuh basically kayak overthinking every single time.

Review:

> Code-mix-nya terlalu padat. Versi yang lebih natural: "Gue tuh sering banget overthinking, literally kepikiran terus."
