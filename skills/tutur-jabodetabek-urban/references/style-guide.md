# Style Guide

## Jaksel

Use:

- Indonesian grammar as the base.
- English inserts for familiar urban/corporate concepts.
- Lightly polished rhythm.

Examples:

- "Basically ini udah oke, tinggal adjust copy dikit biar tone-nya more friendly."
- "Vibe-nya udah dapet, tapi CTA-nya masih kurang clear."

Avoid:

- every word in English,
- fake accent spelling,
- class mockery.

## Jaktim/Jakarta Casual

Use:

- direct phrasing,
- `gue/lo`,
- `nih`, `aja`, `dong`, `gas`, `beres`,
- practical rhythm.

Examples:

- "Gue udah cek, tinggal beresin dikit. Nanti gue kabarin kalau udah kelar."
- "Kalau mau gas hari ini, file finalnya kirim sekarang ya."

Avoid:

- making it rude by default,
- overusing Betawi forms without context.

## Bekasi

Use:

- casual commuter/practical tone,
- `otw`, `macet`, `nanti gue update`,
- direct but not harsh wording.

Examples:

- "Gue udah otw, agak ketahan macet. Nanti gue update kalau udah deket."
- "Aman, gue cek dulu abis ini. Kalau udah beres gue kabarin."

Avoid:

- "planet Bekasi" jokes unless the user explicitly wants meme tone,
- claiming one Bekasi dialect for everyone.

## Neutral Jabodetabek

Use:

- everyday informal Indonesian,
- light Jakarta particles,
- clear meaning.

Example:

> Aku udah cek file-nya. Tinggal revisi bagian harga, habis itu bisa langsung dikirim.
