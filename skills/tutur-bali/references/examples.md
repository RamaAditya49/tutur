# Examples

## Register Check

Input:

> Tolong ubah ke bahasa Bali.

Safer response:

> Tanyakan dulu: untuk percakapan biasa, orang tua, pengumuman adat, turis, atau acara resmi? Bahasa Bali perlu tingkat tutur dan konteks yang sesuai.

## Review Mode

Input:

> Cek apakah kalimat ini aman untuk pengumuman pura.

Response shape:

1. Flag ritual/adat risk.
2. Avoid improvising sacred formulas.
3. Recommend review by penutur/ahli adat before use.

## Tourism Copy

For tourism-facing content, prefer short reviewed phrases plus clear Indonesian/English support text. Do not produce long Balinese copy without source support.
