# Style Guide

## Identity

- User-facing label: Keraton Jogja.
- More precise label: Keraton Kasultanan Yogyakarta.
- Register name: Basa Bagongan.

## Key Distinction

Yogyakarta uses `Basa Bagongan`; Surakarta uses `Basa Kedhaton`.

Basa Bagongan is associated with respect and court identity, but it is not simply high krama. Scholarly summaries describe it as more plain and more leveling than Surakarta Kedhaton in some uses.

## Tone

- Respectful.
- Plain.
- Courtly without being too ornate.
- Less hierarchical than Surakarta Kedhaton.
- Avoid modern slang, marketing language, and bureaucratic Indonesian.

## Grammar Signals

Use with source awareness:

- `manira`: I / first-person marker.
- `pakenira`: you / second-person marker.
- Krama vocabulary with some ngoko-like morphology when appropriate.

Avoid:

- Treating Bagongan as generic krama inggil.
- Using too many Surakarta-style role-specific pronouns.
- Making the output sound like old manuscript language unless requested.

## Morphology Note

Bagongan may mix krama lexical choices with ngoko-like affixes. This is a feature, not automatically an error. Use it sparingly because overuse can look like fake Javanese.

## Safety

Ask for source text or target role if:

- The text is a palace ceremony.
- The audience includes named royal family members.
- The user requests ritual formulas.
- The text will be published as authentic Yogyakarta court language.

Otherwise, provide usable output with a short caveat.
