# Usage Patterns

## Ordinary Output

For cultural event copy, short greetings, and respectful Yogyakarta court flavor, provide adapted text directly. Add a note only when the user asks for ceremony, protocol, inscription, or historical precision.

## Sample Tasks

- Adapt an Indonesian welcome line into Bagongan-inspired tone.
- Review whether text is Bagongan or generic krama.
- Produce light, Bagongan, and ceremony-aware variants.
- Remove Surakarta Kedhaton markers from a Jogja target text.
- Explain why `manira` or `pakenira` should or should not be used.

## Register And Semantics

- Light: restrained Jogja court flavor without dense special vocabulary.
- Bagongan: uses local markers such as `manira`/`pakenira` only when appropriate.
- Ceremony-aware: support text only; mark protocol-sensitive parts.
- Comparative: explicitly name Jogja/Yogyakarta versus Solo/Surakarta differences.

## Review Checklist

- Is the target Yogyakarta/Jogja rather than Surakarta/Solo?
- Are Bagongan markers used sparingly and correctly?
- Does the text avoid generic krama inggil over-polish?
- Does it avoid claiming palace authenticity?
- Are names, titles, dates, and facts unchanged?

## Output Patterns

- Direct request: adapted text first.
- Options request: light, Bagongan, and ceremony-aware variants.
- Review: list Solo/Jogja mixing, generic krama risk, and source gaps.
