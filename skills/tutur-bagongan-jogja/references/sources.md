# Sources

Use these sources as the initial research base. This skill is conservative because Basa Bagongan is specialized and culturally specific.

## Core Source

- UGM Program Studi Bahasa, Sastra, dan Budaya Jawa: "Seputar Jawa: Basa Kedhaton dan Basa Bagongan, Ragam Tutur Khas Keraton Surakarta dan Yogyakarta"
  - https://sastrajawa.fib.ugm.ac.id/seputar-jawa-basa-kedhaton-dan-basa-bagongan-ragam-tutur-khas-keraton-surakarta-dan-yogyakarta/
  - Key use: distinguishes Yogyakarta `Basa Bagongan` from Surakarta `Basa Kedhaton`; notes the `manira` / `pakenira` pair, mixed morphology, and limited special vocabulary.

## Court Context

- Kraton Jogja official site: https://www.kratonjogja.id/
  - Key use: general Yogyakarta palace context.

- Jogja Cagar Budaya: Pura Pakualaman
  - https://jogjacagar.jogjaprov.go.id/detail/1084/puro-pakualaman
  - Key use: confirms Pakualaman as a living court/cultural site in Yogyakarta, distinct from Keraton Kasultanan Yogyakarta.

## General Language References

- KBBI Daring: https://kbbi.kemdikbud.go.id/
- EYD V: https://ejaan.kemdikbud.go.id/

Use KBBI/EYD only for Indonesian support text and explanatory notes. They are not enough to validate Basa Bagongan output.

## Editorial Position

This skill can create respectful, Bagongan-inspired drafts. It should not claim to produce authoritative palace protocol, ritual speech, or historically exact manuscript language.
