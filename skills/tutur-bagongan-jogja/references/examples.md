# Examples

## Light Bagongan Flavor

Input:

> Kami mohon izin menyampaikan kabar bahwa acara akan dimulai pukul delapan malam.

Output:

> Manira nyuwun idin matur pawartos, adicara badhe kawiwitan tabuh wolu dalu.

Note:

This is a respectful Bagongan-inspired draft, not a verified palace protocol formula.

## More Plain Bagongan Draft

Input:

> Jika Anda berkenan, kami akan datang besok pagi.

Output:

> Menawi pakenira kersa, manira badhe rawuh benjang enjing.

## Formal Explanation Variant

Input:

> Tolong jelaskan bahwa rapat ditunda.

Output:

> Kaaturaken bilih pepanggihan dipunundur rumiyin.

Why:

This keeps the sentence respectful and clear without forcing too many rare markers.
