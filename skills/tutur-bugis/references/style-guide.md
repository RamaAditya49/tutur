# Style Guide

## Language Identity

- Name: Bahasa Bugis
- Region/context: Sulawesi Selatan dan komunitas Bugis diaspora; termasuk variasi lokal seperti Soppeng, Bone, Wajo, Sidenreng Rappang, Sinjai, dan daerah Bugis lainnya

## Safety Rules

- Do not invent vocabulary.
- Do not assume one dialect represents all speakers.
- Do not use the language for sacred, ritual, legal, or official text without review.
- Preserve Indonesian terms when the regional equivalent is unknown.
- Ask for a user-provided sample when voice or dialect matters.
- Ask for the target Bugis area/community when the output is public or formal.
- Use Lontara only when the user supplies source text or asks for script-specific handling and sources are available.
- Keep policy, billing, legal, medical, school, and government details in Indonesian unless verified Bugis wording is supplied.

## Practical Use

Use this skill for:

- source-aware regional-language output,
- tone adaptation,
- localized greetings or short public text,
- review of AI-generated local-language output,
- identifying uncertainty before publication.

## Registers

### Everyday/Casual

Use short Bugis cues with clear Indonesian:

- `Tabe'`
- `Iye`
- `Basa Ugi`

Suitable for casual captions, simple invitations, and learning prompts.

### Polite/Public

Use politeness markers with restraint:

> Tabe', Bapak/Ibu. Kami mengundang idi' hadir pada kegiatan komunitas hari Sabtu pukul 19.30.

Review public text with a competent speaker before printing or broadcasting.

### Professional/Admin

For admin, school, sales, CS, finance, tourism, or government-adjacent work:

- keep operational facts in Indonesian,
- use Bugis only as checked local warmth,
- avoid full translation without user sample.

### Adat/Ceremonial

Do not generate adat, wedding, religious, or ceremonial Bugis text from scratch. Ask for source text from the community and use this skill only to review or flag uncertainty.
