# Local Mirror

## Source Mirror

- Register: Surakarta/Solo court-inspired Basa Kedhaton.
- Court context: Kasunanan Surakarta and related Surakarta court culture; Mangkunegaran is related but not automatically identical.
- Main distinction: keep this separate from Yogyakarta Basa Bagongan and ordinary Bahasa Jawa.
- Local references include the repo's `sources.md`, `style-guide.md`, `lexicon.md`, and examples.
- General KBBI/EYD references are useful only for Indonesian support text, not for validating Kedhaton wording.

## Local Usability

Use this file before browsing. The skill can produce respectful Surakarta-leaning court-inspired wording for ordinary cultural copy, event flavor, and review.

It must not claim authoritative palace protocol, ritual validity, inscription accuracy, or manuscript-level authenticity. For those contexts, produce usable support text and mark the exact parts requiring a Surakarta court-language expert.

## Vocabulary And Semantics

- Basa Kedhaton is role-sensitive; pronouns, honorifics, and address choices depend on speaker/listener status.
- Krama structure and formal Javanese morphology can support the tone, but generic krama is not automatically Kedhaton.
- Use special vocabulary only when it appears in the local lexicon or a user-provided source phrase.
- Avoid importing Yogyakarta Bagongan markers such as `manira` and `pakenira` unless the task is comparative.
- Avoid direct first/second-person pronouns when status is unclear.

## Generation Rules

1. Identify whether the user wants light flavor, formal court-inspired output, or ceremony-aware support text.
2. Preserve Indonesian meaning and factual details.
3. Use verified Kedhaton markers sparingly and consistently.
4. Prefer pronoun-light phrasing when speaker/listener status is unclear.
5. Add a review note for ceremony, protocol, ritual, inscription, or manuscript contexts.

## Starter Lexicon Policy

Use `lexicon.md` as the only local starter list. Do not invent palace vocabulary. Do not expand the lexicon without a reliable source or expert-reviewed sample.
