# Style Guide

## Identity

- User-facing label: Keraton Solo.
- More precise label: Keraton Kasunanan Surakarta / Surakarta court.
- Register name: Basa Kedhaton.

## Key Distinction

Surakarta uses `Basa Kedhaton`; Yogyakarta uses `Basa Bagongan`. Keep them separate.

Surakarta Kedhaton tends to preserve krama sentence structure more strongly than Yogyakarta Bagongan. It can use special pronouns and court vocabulary, but those choices are tied to social role and rank.

## Tone

- Respectful.
- Restrained.
- Formal but not flowery.
- More hierarchical than Yogyakarta Bagongan.
- Avoid slang, modern promotional language, and casual Indonesian.

## Grammar Signals

Use with source awareness:

- Passive prefix: `dipun-`
- Formal suffix: `-aken`
- Literary/Kawi-flavored forms when known and appropriate.

Avoid:

- Randomly adding archaic words.
- Mixing ngoko suffixes into Surakarta Kedhaton unless the source requires comparison.
- Using Yogyakarta Bagongan as a default.

## Pronouns

Surakarta Kedhaton pronouns can vary by role. Known examples from scholarly summaries include:

- `mara` / `para`
- `manira` / `pakenira`
- `kula` / `jengandika`
- `robaja` / `panten`

Use these only when the speaker/listener role is clear. If unclear, avoid direct pronouns by restructuring the sentence.

## Safety

Ask for source text or target role if:

- The text is a palace ceremony.
- The audience includes named royal family members.
- The user requests ritual formulas.
- The text will be published as authentic Surakarta court language.

Otherwise, provide usable output with a short caveat.
