# Usage Patterns

## Ordinary Output

For cultural event copy, short greetings, and respectful tone adaptation, provide the adapted text directly. Add a note only when the user asks for ceremonial, protocol, inscription, or historical precision.

## Sample Tasks

- Adapt an Indonesian welcome line into Surakarta court-inspired tone.
- Review whether text sounds like generic krama or Kedhaton-flavored wording.
- Produce light, formal, and ceremony-aware variants.
- Remove Yogyakarta Bagongan markers from a Solo target text.
- Make the wording pronoun-light because speaker status is unclear.

## Register And Semantics

- Light: mostly Indonesian/Javanese support text with restrained court flavor.
- Formal: more krama structure and vetted Kedhaton markers.
- Ceremony-aware: support text only; mark protocol-sensitive parts.
- Comparative: explicitly name Solo/Surakarta versus Jogja/Yogyakarta differences.

## Review Checklist

- Is the target Surakarta/Solo rather than Yogyakarta/Jogja?
- Are pronouns appropriate or avoided?
- Are Kedhaton markers sourced from local lexicon or user sample?
- Does the text avoid claiming palace authenticity?
- Are names, titles, dates, and facts unchanged?

## Output Patterns

- Direct request: adapted text first.
- Options request: light, formal, and ceremony-aware variants.
- Review: list Solo/Jogja mixing, generic krama risk, and source gaps.
