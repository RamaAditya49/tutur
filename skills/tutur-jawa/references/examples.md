# Examples

## Target-Language Examples

Ngoko/everyday:

> Ayo padha njaga basa Jawa.

Meaning:

> Mari kita menjaga bahasa Jawa.

Krama/public:

> Mangga sami nguri-uri basa Jawi.

Meaning:

> Mari bersama-sama melestarikan bahasa Jawa.

Note:

Use ngoko for familiar contexts and krama for public, elder-facing, or formal contexts.

## Register Check

Input:

> Tolong ubah ke bahasa Jawa.

Safer response:

> Tanyakan dulu: untuk teman sebaya, orang tua, pengumuman umum, atau acara resmi? Bahasa Jawa perlu tingkat tutur yang sesuai.

## Review Mode

Input:

> Cek apakah kalimat ini terlalu ngoko untuk undangan.

Response shape:

1. Identify the current level.
2. Flag ngoko or mixed-level choices.
3. Provide a safer krama/public version, with a caveat if uncertain.

## Dialect Caution

For Banyumasan, Suroboyoan, Osing, Tengger, Cirebonan, or diaspora Javanese, ask for local target before rewriting.
