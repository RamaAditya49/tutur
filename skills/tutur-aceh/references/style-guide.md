# Style Guide

## Language Identity

- Name: Bahasa Aceh
- Region/context: Aceh; also used by Acehnese communities outside Aceh
- Glottocode: `achi1257`
- ISO 639-3: `ace`

## Register Position

Bahasa Aceh is not Indonesian with local slang. Do not "Aceh-ify" Indonesian by adding a few local words around Indonesian grammar and claiming it as correct Acehnese.

Use Bahasa Aceh output only when:

- the wording is sourced,
- the user provides a sample,
- or the user accepts a source-aware working output with uncertain terms marked.

## Safety Rules

- Do not invent vocabulary.
- Do not assume one dialect represents all speakers.
- Do not force Acehnese spelling when the source gives only Indonesian.
- Do not use the language for sacred, ritual, legal, or official text without review.
- Preserve Indonesian terms when the regional equivalent is unknown.
- Ask for a user-provided sample when voice or dialect matters.

## Practical Use

Use this skill for:

- source-aware regional-language output,
- tone adaptation,
- localized greetings or short public text,
- review of AI-generated local-language output,
- identifying uncertainty before publication.

## Practical Notes

- Use KBDA Daring for word lookup before replacing important terms.
- Keep Islamic greetings and religious phrases as the user writes them unless the user asks for localization.
- If the target audience is a specific part of Aceh, ask for the dialect/community when the phrasing matters.
