# Roadmap

## Phase 1

- Publish `tutur-humanizer`.
- Keep the skill broad enough for official, professional, casual, gaul, and sales contexts.
- Include official source references and a small scanner script.

## Phase 2

- Add specialist skills:
  - `tutur-resmi`
  - `tutur-gaul`
  - `tutur-korporat-profesi`
  - `tutur-jabodetabek-urban`
- Add initial regional rollout skills:
  - `tutur-aceh`
  - `tutur-gayo`
  - `tutur-minangkabau`
  - `tutur-sunda`
  - `tutur-jawa`
  - `tutur-madura`
  - `tutur-bali`
  - `tutur-sasak`
  - `tutur-banjar`
- Add court-register skills:
  - `tutur-kedhaton-solo`
  - `tutur-bagongan-jogja`
  - `tutur-mangkunegaran`
  - `tutur-pakualaman`
  - `tutur-cirebon`

## Phase 3

- Add examples from real anonymized writing patterns.
- Add a larger Indonesian phrase scanner.
- Add optional presets for brand voice, document type, and audience.
- Expand regional and court registers with stronger source material and expert-reviewed examples.
- Split the umbrella corporate-register skill into role-specific skills when the resource base is richer:
  - `tutur-sales`
  - `tutur-customer-service`
  - `tutur-admin`
  - `tutur-hrd`
  - `tutur-finance-admin`
  - `tutur-marketing`
- Split the urban Jabodetabek sociolect skill when better local examples are available:
  - `tutur-jaksel`
  - `tutur-jaktim`
  - `tutur-bekasi`
  - `tutur-betawi-jakarta`

## Non-Goals

- Tutur is not a grammar checker only.
- Tutur is not meant to force all writing into formal Indonesian.
- Tutur should not invent facts, citations, or claims while rewriting.
- Tutur should not claim authentic ritual or palace-protocol language without verified sources.
